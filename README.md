# solucion-al-calentamiento-global
en este chatbot consiste en brindar una posible solucion o aporte para mitigar el calentamiento global asi como concientizar sobre que es, que lo genera y que consecuencias tiene sobre nosotros

# funciones
este chat bot tendria la funcion de responder las preguntas ya mencionadas con los siguientes comandos

-informacion
este comando se usa para que el chatbot nos brinde informacion sobre que es el calentamiento global para que asi empecemos por tener una idea basica sobre que es el ya mencionado calentamiento global

-consecuancias
este comando se usa para que el chatbot nos brinde informacion cuales son las consecuencias que tiene el calentamiento global sobre nosotros y el planeta en general ya que este problema tiene repercucion en multiples ambitos 

-causas
este comando se usa para que el chatbot nos brinde informacion sobre que genera el calentamiento global y de como podemos mitigar su progreso para asi contribuir por el bienestar de todos

-!
este prefijo se usa para la activacion del chatbot
